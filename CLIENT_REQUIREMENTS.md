## Design Requirements

### 1. Design Category Information
- **Category Type** (Required)
  - Kitchen Designs
  - Wardrobe Designs
  - Bathroom Designs
  - Master Bedroom Designs
  - Living Room Designs
  - Pooja Room Designs
  - TV Unit Designs
  - False Ceiling Designs
  - Kids Bedroom Designs
  - Dining Room Designs
  - Foyer Designs
  - Home Office Designs
  - Study Room Designs
  - Balcony Designs
  - Guest Bedroom Designs
  - And other design categories

### 2. Design Basic Information
- **Design Title/Name** (Required)
  - A descriptive name for the design (e.g., "Modern Modular Kitchen with Island")
  
- **Design Description** (Required)
  - Short description (1-2 sentences)
  - Long/Detailed description (paragraph format)
  
- **Design Style** (Required)
  - Modern
  - Contemporary
  - Traditional
  - Minimalist
  - Luxury
  - Scandinavian
  - Industrial
  - Classic
  - Other styles

### 3. Design Pricing
- **Price** (Required)
  - Price in Indian Rupees (INR)
  - Format: e.g., "₹4,50,000" or "₹3,20,000"
  - Price range option: e.g., "20 - 30 Lakhs"
  
- **Price Category** (Optional)
  - Budget-friendly
  - Mid-range
  - Premium
  - Luxury

### 4. Design Dimensions & Size
- **Area** (Required)
  - Area in square feet (e.g., "120 sq ft", "80 sq ft")
  - Dimension format: e.g., "12x10 feet"
  
- **Room Dimensions** (Optional)
  - Length (in feet)
  - Width (in feet)
  - Height (in feet)

### 5. Design Features
- **Features List** (Required)
  - Array of key features (e.g., "Central island", "Breakfast counter", "Premium finishes", "Smart storage")
  - Minimum 3-5 features per design

### 6. Design Specifications
- **Color Scheme** (Required)
  - Primary colors used
  - Color palette description (e.g., "White, Gray, Wood")
  
- **Materials** (Required)
  - List of materials used (e.g., "Laminate, Granite, Stainless Steel")
  - Material types:
    - Carcass material (BWR Plywood, MDF, HDF)
    - Shutter material (Laminate, Membrane, Acrylic)
    - Countertop material (Granite, Quartz, Marble)
    - Hardware specifications
  
- **Furniture/Components** (Required)
  - List of furniture pieces (e.g., "Modular Cabinets, Island, Breakfast Counter")
  
- **Lighting** (Required)
  - Lighting types (e.g., "LED Under-cabinet, Pendant Lights")
  
- **Additional Specifications** (Optional)
  - Appliances included
  - Special features
  - Customizations

### 7. Design Sections
- **Detailed Sections** (Optional but Recommended)
  - Each section should have:
    - Section title (e.g., "Kitchen Layout", "Storage Solutions", "Premium Features")
    - List of items/points under each section

### 8. Design Ideal For
- **Target Audience** (Optional)
  - Description of who this design is ideal for (e.g., "Cooking enthusiasts and families who love to entertain")

## Image Requirements

### 1. Design Images

#### Primary Image (Required)
- **Main Design Image**
  - High-resolution image (recommended: 800px width minimum)
  - Format: JPG, PNG, WebP
  - Aspect ratio: 16:9 or 4:3 recommended
  - File size: Optimized for web (under 500KB recommended)
  - URL or file upload

#### Image Gallery (Required)
- **Multiple Design Images**
  - Minimum 3-5 images per design
  - Recommended: 5-10 images
  - Each image should have:
    - Full-size URL (800px+ width)
    - Thumbnail URL (200px width)
    - Alt text/description
    - Category/tag (e.g., "overview", "island", "storage", "detail")
  
- **Image Categories** (Optional)
  - Overview images
  - Detail shots
  - Room-specific images
  - Feature highlights

### 2. Project Images

#### Primary Project Image (Required)
- **Featured Image**
  - High-resolution (500x400px minimum)
  - Represents the project
  - Format: JPG, PNG, WebP

#### Project Gallery (Required)
- **Project Image Collection**
  - Minimum 3-6 images per project
  - Recommended: 6-15 images
  - Should include:
    - Different rooms/areas
    - Before/after shots (if available)
    - Detail shots
    - Full room views
  
- **Image Metadata** (Optional)
  - Room/area category (living-room, kitchen, bedroom, etc.)
  - Is primary image flag
  - Upload date

### 3. Category/Thumbnail Images
- **Category Display Image** (Required for categories)
  - 400x300px recommended
  - Used in category listings
  - Format: JPG, PNG, WebP

### 4. Component Images
- **Kitchen Component Images** (For kitchen components)
  - Cabinet images
  - Finish samples
  - Handle designs
  - Multiple angles/views

### 5. Image Quality Standards
- **Resolution Requirements**
  - Minimum: 800px width for detail pages
  - Recommended: 1200px+ width for high-quality display
  - Thumbnails: 200-300px width
  
- **File Formats**
  - Primary: JPG (for photos)
  - Secondary: PNG (for graphics with transparency)
  - Modern: WebP (for optimized web delivery)
  
- **File Size Optimization**
  - Compressed for web delivery
  - CDN-optimized URLs preferred
  - Automatic resizing support recommended

### 6. Image Organization
- **Naming Convention** (Recommended)
  - Descriptive filenames
  - Category prefixes (e.g., "kitchen-", "wardrobe-")
  - Sequential numbering
  
- **Image Storage**
  - Cloud storage URLs (CDN preferred)
  - Organized by category/project
  - Backup storage recommended

## Data Submission Format

### Recommended Data Structure

#### For Designs
```json
{
  "id": "unique-design-id",
  "name": "Design Name",
  "category": "kitchen",
  "style": "Modern",
  "price": "₹4,50,000",
  "area": "120 sq ft",
  "description": "Short description",
  "longDescription": "Detailed description",
  "image": "primary-image-url",
  "images": [
    {
      "url": "image-url",
      "thumbnail": "thumbnail-url",
      "alt": "Image description",
      "category": "overview"
    }
  ],
  "features": ["feature1", "feature2"],
  "specifications": {
    "dimension": "12x10 feet",
    "color": "White, Gray",
    "materials": "Laminate, Granite",
    "furniture": "Modular Cabinets",
    "lighting": "LED Under-cabinet"
  }
}
```

#### For Projects
```json
{
  "id": "project-id",
  "title": "Project Title",
  "location": "Location, City",
  "scope": "Full Home, Kitchen",
  "bhk": "3-BHK",
  "pricing": "20 - 30 Lakhs",
  "area": "1200 sq ft",
  "status": "COMPLETED",
  "image": "primary-image-url",
  "images": ["image1-url", "image2-url"],
  "style": "Contemporary",
  "features": ["feature1", "feature2"]
}
```

## Notes & Guidelines

1. **Image Quality**: All images should be high-resolution, properly optimized, and web-ready
2. **Data Consistency**: Maintain consistent formatting for prices, dimensions, and specifications
3. **Required Fields**: Mark clearly which fields are required vs. optional
4. **Validation**: Implement proper validation for all input fields
5. **File Formats**: Support common image formats (JPG, PNG, WebP)
6. **Storage**: Use CDN or cloud storage for images
7. **Metadata**: Include proper alt text and descriptions for all images
8. **Organization**: Organize data by category and maintain proper naming conventions
